# area-check · 平面面積計算器

拍照 → 直接顯示面積。自動偵測照片下方的尺(整支 = 15 cm)換算比例,
並自動找出所有用黑色/藍色畫出的封閉框線,分別計算框內實際面積(cm²),
再依選擇的植髮密度(30–90 株/cm²)估算所需植髮株數。

## 使用方式
1. 開啟 `index.html`(或線上版),選擇/拍攝照片:下方放一把 15 cm 的尺,
   待測區域用黑或藍色畫出封閉框線,與待測物同一平面、盡量正對拍攝。
2. 載入後自動顯示每個框的面積、總面積與植髮株數。

## 手動輔助(需要時才用)
- 自動偵測不到尺、或尺長不準:點「手動標尺」,在尺上點兩個已知距離的點。
- 沒抓到的框:直接點框內部補偵測;點已偵測的框可移除。
- 尺長預設 15 cm,可於上方輸入框修改。

純前端單一 HTML 檔,無需安裝、無外部相依。
