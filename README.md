# area-check · 平面面積計算器

用一把已知長度的尺（例如 15 cm）做「像素→公分」比例校正，再框選一個或多個區域，
用 Shoelace 公式分別計算各區域的實際面積（cm² / mm² / m²）。

## 使用方式
1. 用瀏覽器開啟 `area-calculator.html`（或線上版 `index.html`）。
2. 載入照片（下方放一把尺，與待測物同一平面，盡量正對拍攝）。
3. 「標記尺的兩端」做校正（預設 15 cm，可修改）。
4. 「開始框選」沿邊界點一圈 →「完成此框」；可重複新增多個框，各自顯示面積與總和。

純前端單一 HTML 檔，無需安裝、無外部相依。
